# Using Web PKI BETA

```html
<script
  src="https://getwebpkibeta.lacunasoftware.com/Scripts/LacunaWebPKI/lacuna-web-pki-beta-2.10.1.js"
  integrity="sha256-+jltl/hwRRlwr7OCt41kPlSaEnDm2marX++KuNAiLMY="
  crossorigin="anonymous"></script>
```
