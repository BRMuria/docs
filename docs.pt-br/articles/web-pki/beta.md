# Utilizando o Web PKI BETA

```html
<script
  src="https://cdn.lacunasoftware.com/libs/web-pki-beta/lacuna-web-pki-beta-2.12.5.min.js"
  integrity="sha256-18X+Nop7+NxZnj1+ItS8pjm3F6anqxANcsu5K3wJ9CA="
  crossorigin="anonymous"></script>
```
